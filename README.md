# Data Analysis Capstone – TripleTen

## Descripción
Proyecto final del Bootcamp Data Analyst de TripleTen, enfocado en resolver problemas reales de negocio utilizando análisis de datos, SQL y estadística.

## Proyectos incluidos
1. Telecom Analytics – Call Me Maybe
2. SQL Analysis – Bookstore Database
3. Marketing Analytics – A/B Test

## Tecnologías
Python, Pandas, NumPy, SQL, Matplotlib, Seaborn, SciPy

## Enfoque
- Análisis exploratorio
- KPIs
- Visualización
- Toma de decisiones basada en datos
